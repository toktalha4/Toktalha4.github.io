# Hair Time Studio v3

Bu sürüm premium / lüks hissi vermesi için sıfırdan güçlendirildi.

## Değiştirmen gereken ana yerler

### index.html
- Hair Time Studio adını gerçek işletme adıyla değiştir
- `+90 5XX XXX XX XX` alanlarını gerçek telefonla değiştir
- `https://wa.me/905000000000` içeren tüm linkleri gerçek WhatsApp numarasıyla değiştir
- `Buraya açık adres gelecek` alanını gerçek adresle değiştir
- `https://instagram.com/kullaniciadi` linkini gerçek Instagram ile değiştir
- `https://maps.google.com` linklerini gerçek Google Maps konum linkiyle değiştir
- Yorumlar bölümünü gerçek Google yorumlarıyla değiştir
- Galerideki örnek görselleri kendi gerçek fotoğraflarınla değiştir

### script.js
- İçindeki `905000000000` numarasını gerçek WhatsApp numarasıyla değiştir

## Telefonda güncelleme
1. Bu klasördeki `index.html`, `styles.css`, `script.js` dosyalarını GitHub repona yükle
2. Eski dosyaların üstüne yaz
3. Commit at
4. 1-3 dakika bekle
5. Siteyi yenile
